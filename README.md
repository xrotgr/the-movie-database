# Movies Website

This is a single page application made with React + TypeScript, Tanstack router for routing, Tanstack Query for handling server state. UI is implemented using Shadcn UI and Tailwind CSS.
It is a website, where you can see information about different movies and TV shows. There is a homepage with a carousel, pages with different kinds of movies, page of each individual movie or TV show. Also I made functionalities of infinite scroll and filtering.

## Installation

1. Clone the repository:

```bash
 git clone https://github.com/xrotgr/the-movie-database.git
```

2. Install dependencies:

```bash
 npm install
```

## Usage

To run the project, use the following command:

```bash
npm run dev
```
